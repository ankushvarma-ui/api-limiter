Request Flow
1

Request Arrives

Client hits any endpoint. Middleware intercepts before reaching any route handler.

2

IP Block Check

MongoDB queried for active block on this IP. If blocked → 403 immediately returned.

3

Rate Window Check

In-memory sliding window counter checked. Over 100 req/15min → 429 returned.

4

Abuse Detection

Violation counter incremented. After 3 violations → IP blocked for 1 hour.

5

Pass Through + Log

Request proceeds to route handler. On finish, logged to MongoDB with status code.

200 OK — /protected-data✓ Authorized
{ "success": true, "data": { "secretInfo": "...", "user": { "username": "alice" } } }
429 Too Many Requests⚡ Rate Limited
{ "success": false, "message": "Too many requests", "retryAfter": "2026-03-09T..." }
403 Forbidden🚫 IP Blocked
{ "success": false, "message": "IP temporarily blocked", "retryAfterMinutes": 58 }
401 Unauthorized🔑 No Token
{ "success": false, "message": "Access denied. No token provided." }
